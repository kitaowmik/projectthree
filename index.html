I attempted to create a solar system with this project. My vision was definitely more ambitious than how this turned out, and there are a lot of things I would like to improve on here. I used the GLTFloader and FlyControls add-ons for this project. The loader allowed me to put the planets in, which I had created through Blender. The FlyControls add-on will enable you to move around like you're in a spaceship while navigating this setting, which I think suits this environment. This world is quite sci-fi/"Mass Effect"-inspired. It was an attempt to create a world where space appeared inhabitable, and other planets were almost like continents that could be easily traveled to. In the future, I would like to add more planets and work on texture-painting them to suit my ideas.
